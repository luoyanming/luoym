今天项目中用到正整数的验证，乘着余热记录下我们在项目中常用的一些js正则表达式验证。
<!-- more -->

## 1、`Code`
```javascript
// func
function checkInput(string) {
  var reg = '此处写正则表达式';
  
  return reg.test(string);
}
```

## 2、常用正则表达式
```javascript
// 正整数
var reg = /^\+?[1-9][0-9]*$/;

// 大于0的数字(包括整数和小数)
var reg = /^(?!(0[0-9]{0,}$))[0-9]{1,}[.]{0,}[0-9]{0,}$/;

// 大于等于0的数字
var reg = /^\d+(\.\d+)?$/;

// 身份证
var reg = /(^\d{15}$)|(^\d{18}$)|(^\d{17}(\d|X|x)$)/;

// 手机号
var reg = /^1\d{10}$/;

// 判断链接
var reg = /(http:\/\/[\w.\/]+)(?![^<]+>)/gi;
```
