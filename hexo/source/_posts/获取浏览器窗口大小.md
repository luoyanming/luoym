获取浏览器窗口大小的兼容方法。
<!-- more -->

## 1、获取浏览器可视区域大小
```javascript
var pageWidth = window.innerWidth,
    pageHeight = window.innerHeight;

if(typeof pageWidth != 'number') {
    if(document.compatMode == 'CSS1Compat') {
        // 标准模式下
        pageWidth = document.documentElement.clientWidth;
        pageHeight = document.documentElement.clientHeight;
    } else {
        pageWidth = document.body.clientWidth;
        pageHeight = document.body.clientHeight;
    }
}
```

## 3、`location` 对象
```javascript
// 返回URL中的hash（#后跟零或多个字符，若没有，返回空字符串）
console.log(location.hash) // #first

// 返回服务器名称和端口号（如果有端口号）
console.log(location.host) // www.luoym.com:4000

// 返回不带端口号的服务器名称
console.log(location.hostname) // www.luoym.com

// 返回当前页面的完整 url
console.log(location.href) // hhtp://www.luoym.com

// 返回url中的目录和、或文件名
console.log(location.pathname) // /archives/  或  /2017/07/26/Javascript%E5%AD%A6%E4%B9%A0002%20--%20array/

// 返回端口号（若没有，返回空字符串）
console.log(location.port) // 4000

// 返回页面使用的协议
console.log(location.protocal) // http: 或者 https:

// 返回url的以？开头的查询字符串
console.log(location.search) // ?id=1201
```

## 4、`navigator` 对象
```javascript
// 浏览器名称
navigator.appCodeName // Mozilla

// 完整的浏览器名称
navigator.appName // Netscape

// 浏览器版本
navigator.appVersion // 5.0 (Windows NT 6.1; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/59.0.3071.115 Safari/537.36

// 浏览器是否连网络
navigator.onLine // true

// 浏览器所在系统平台
navigator.platform // win32

// 浏览器的用户代理字符串
navigator.userAgent // Mozilla/5.0 (Windows NT 6.1; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/59.0.3071.115 Safari/537.36
```
