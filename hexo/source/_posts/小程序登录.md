&nbsp;

## 1、首先，去微信登录
登录API：[`wx.login`](https://developers.weixin.qq.com/miniprogram/dev/api/api-login.html)
```javascript
wx.login({
  success: res => {
    if (res.code) {
      // 登陆成功，拿到临时登录凭证code
      doLogin(res.code);
    }
  }
})
```
&nbsp;

## 2、将`code`值传给服务端
```javascript
doLogin: function(code) {
    wx.request(....)
    // 发送code给服务端，用来换取 用户唯一标识openid 和 会话密钥session_key
}
```
