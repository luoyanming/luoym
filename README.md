## HEXO
传送门 [https://hexo.io/zh-cn/docs/]

## 启动
```vim

hexo server

// 访问网址为： http://localhost:4000/

```

## 生成静态文件
```vim

hexo generate

// or

hexo g


```

## 部署
```vim

hexo deploy

// or

hexo d

```

## 清除缓存文件 (db.json) 和已生成的静态文件 (public)
```vim

hexo clean

```

## 列出网站资料
```vim

hexo list <type>

// type arguments: page, post, route, tag, category

```
